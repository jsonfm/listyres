### 📄🥙 Listyres

A simple application for listing restaurants.

### 👀 Live Demo

[here](https://jsonfm.github.io/listyres/)

### ⛏️ Technologies

-   React.js
-   Tailwind
-   Python

### 📦 Installation

```
npm install
```

### ⚡️ Development

```
npm run dev
```

### 🚀 Deployment

```
npm run deploy
```
